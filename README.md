# Weather App ☀️🌧️

A simple and clean Flutter weather app using BLoC for state management and API.

## 🔥 Features
- Search for any city 🌍
- Displays:
  - City name
  - Current time
  - Temperature
  - Max/Min temperature
  - Weather condition (e.g. sunny, cloudy)

## 🧠 State Management
- BLoC (Cubit pattern)

## 🛠️ Built With
- Flutter
- BLoC (flutter_bloc)
- Weather API
